# CyberEd Range - Educational Cyber Range Platform

An interactive cybersecurity learning platform designed for high school students, inspired by Hack The Box but tailored for educational purposes.

## Overview

CyberEd Range provides hands-on cybersecurity challenges across multiple domains, integrating important learning objectives and vocabulary to ensure students build a strong foundation in security concepts.

## Usage

1. **Dashboard**: View overall progress and select challenge categories
2. **Challenges**: Choose and complete challenges within each category
3. **Vocabulary**: Review and study cybersecurity terminology
4. **Progress**: Track completed challenges and earned points

## Challenge Structure

Each challenge includes:
- **Title & Description**: Context and background
- **Difficulty Level**: Easy, Medium, or Hard
- **Points**: Reward for completion
- **Learning Objective**: What you'll learn
- **Vocabulary Terms**: Key concepts highlighted
- **Challenge Prompt**: The actual problem to solve
- **Hints**: Progressive help when needed
- **Answer Validation**: Submit and check solutions
- **Explanation**: Detailed breakdown after completion
