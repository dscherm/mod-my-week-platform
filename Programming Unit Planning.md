# Programming Unit Planning Guide
## Arrays, Loops & Traversal with p5.js

## Course Overview

This comprehensive programming curriculum teaches students fundamental programming concepts through creative coding with p5.js. The course follows the 5E instructional model (Explore, Explain, Apply, Practice, Challenge) to maximize student engagement and learning outcomes through visual, interactive projects.

**Total Duration:** 4 weeks (20 class days)
**Target Audience:** High school students (grades 9-12)
**Prerequisites:** Basic familiarity with variables and functions in p5.js

---

## Week 1: Arrays Basics

### Big Idea
Arrays store collections of values that we can access by index.

### Day 1: Intro to Arrays
**Duration:** 60-75 minutes

**Learning Objectives:**
- Define what an array is and explain its purpose
- Create arrays using square bracket notation
- Access array elements using index notation
- Identify the first and last elements of an array

**Key Vocabulary:**
- Array, Index, Element, Length, Square Brackets, Zero-indexed

**5E Structure:**
| Phase | Activity |
|-------|----------|
| Explore | "Color Palette Challenge" - students try storing 5 colors without arrays |
| Explain | Array syntax, index notation, zero-based indexing |
| Apply | Color Palette Array exercise - create and access arrays |
| Practice | First, Last, Middle exercise - navigate arrays |
| Challenge | Random Word Display - combine arrays with random() |

**Platform Integration:** Exercises w1d1-1, w1d1-2, w1d1-3

## Differentiation Strategies

### For Struggling Students
- Provide starter code with more scaffolding
- Use fill-in-the-blank exercises
- Pair with advanced students
- Focus on core concepts before variations
- Offer additional visual examples
- Allow more hint usage without penalty

### For Advanced Students
- Extension challenges on each exercise
- Independent project proposals
- Peer tutoring opportunities
- Portfolio development
- Advanced physics simulations
- Game jam participation
